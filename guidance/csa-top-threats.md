# CSA Top Threats

## Acronyms, Abbreviations, and Initialisms

| Short Form | Full Form |
| - | - |
| CSA | Cloud Security Alliance |

## Overview

Top Threats is a working group established by the CSA that aims to provide organizations with an up-to-date, expert-informed understanding of cloud security risks, threats and vulnerabilities in order to make educated risk-management decisions regarding cloud adoption strategies.[[ยน]](#ref1)

Notable works from this group include the Pandemic Eleven, the Egregious Eleven, the Treacherous Twelve, and the Notorious Nine.
